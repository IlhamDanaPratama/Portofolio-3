PORTOFOLIO - 3

Rancang Bangun Pakan Ayam Otomatis Berbasis Internet Of Things Menggunakan ESP8266

Tools: SolidWorks, Arduino IDE, Fritzing, Microsoft Excel

Project ini menggunakan ESP8266 sebagai microcontroller, menggunakan rel dengan penggerak motor stepper dan IoT untuk memonitoring jadwal pemberi makan ayam sesuai waktu yang telah disetting. Sistem kerja ketika waktu sudah kita setting maka motor stepper akan menggerakkan rel sampai menyentuh limit switch, kemudian akan kembali ke titik awal. Proses berjalannya sistem akan dimonitoring menggunakan aplikasi Blynk IoT.

Desain Kerangka Prototype Menggunakan SolidWorks: Pada tahap ini, model 3D dari kerangka prototype dirancang menggunakan SolidWorks. Desain ini mencakup detail teknis mengenai struktur mekanik sistem pemberi makan ayam otomatis, termasuk penempatan komponen elektronik, motor stepper, dan rel. SolidWorks digunakan untuk memvisualisasikan dan memastikan bahwa semua komponen dapat berfungsi secara efektif dalam ruang yang tersedia.

Desain Rangkaian Elektronika Menggunakan Fritzing: Pada tahap ini, desain rangkaian elektronika dari sistem dibuat menggunakan Fritzing. Fritzing membantu dalam membuat skema rangkaian yang jelas dan mudah dipahami, termasuk bagaimana sensor, microcontroller, motor, dan komponen lain dihubungkan. Rangkaian ini kemudian diimplementasikan pada breadboard sebelum dipasang pada PCB untuk prototype akhir.

Membuat Script Prototype Menggunakan Arduino IDE: Script yang diperlukan untuk menjalankan sistem dibuat menggunakan Arduino IDE. Script ini mengatur logika kerja dari ESP8266, termasuk pemrograman untuk mengontrol motor stepper, membaca data dari sensor, dan mengirimkan informasi ke aplikasi Blynk IoT. Script juga mencakup pengaturan waktu otomatis untuk pemberian makan ayam sesuai jadwal yang telah ditentukan.

Desain Dashboard Menggunakan Cloud Blynk IoT: Terakhir, desain dashboard untuk memonitoring dan mengendalikan sistem pemberi makan ayam dibuat menggunakan Cloud Blynk IoT. Dashboard ini menampilkan data real-time mengenai status sistem, termasuk waktu pemberian makan, kondisi sensor, dan status motor. Pengguna dapat mengakses dashboard ini melalui smartphone untuk memantau dan menyesuaikan pengaturan sistem dari jarak jauh.
